Basics of the web and HTML


Introduction to the Web






World wide web (WWW) is a collection of HyperText markup Language (HTML) documents.A typical HTML may contain plain texts, images, videos, music, animations e.t.c. The web was invented in the 1990s and has about 30 Billion pages up to date.

The picture avbove shows the structure of the internet. Typically, the browser in our computer communicates with the web server(s) through http requests.

HyperText markup Language (HTML)




HTML Markups

An HTML element refers to everything within a set of opening and closing tags.
text bolding: <b> text </b> text
text itallics (emphasis - em): <em> text </em> text

HTML Attributes

Attribute is a property of an HTML element. For example, to set the href attribute of an anchor tag-
text link (anchor) - <a href: "www.google.com" > text </a> text

Void Tags: images - <img src = "URL" alt = "text">



White Spaces: text break - <br> text paragraph - <p> text </p> Inline v.s. Block

For a block, there is a "box" around the content. For inlines, there are no "boxes" around the content.

Document structure

<p> <DOCTYPE HTML><br> <html> <head> <title> Introduction to Programming!</title> <body> <b>content/b> </body> </head> </html> </p>

The DOCTYPE declaration defines the document type to be HTML.
The html tag describes an HTML document.
The head tag provides information about the document.
The title tag provides a title for the document.
The body tag describes the visible page content.
The head tag describes a heading.
The p tag describes a paragraph.
